LEVEL 1: GRAD-CAM (Gradient-weighted Class Activation Mapping)
--------------------------------------------------------------------------------
Technique: Visual Attention Heatmaps
Purpose: Shows WHICH PARTS of the image the model focuses on

Implementation Details:
✓ Target Layer: ResNet50 layer4[-1] (final convolutional layer)
✓ Method: Combines gradient information with activation maps
✓ Output: Color-coded heatmap overlay (JET colormap)
✓ Transparency: 40% alpha for clear original image visibility
✓ Resolution: Automatically resized to match input image dimensions

How It Works:
1. Forward pass: Image → Model → Prediction
2. Backward pass: Compute gradients for target class
3. Weight activations by gradient importance
4. Generate spatial heatmap showing important regions
5. Overlay on original image with color intensity = importance

Visual Interpretation:
- Red/Yellow regions: High importance (model focused here)
- Blue/Green regions: Low importance
- Shows diseased spots, leaf patterns, or healthy indicators

File: gradcam.py (176 lines)
Features: generate_cam(), visualize(), create_gradcam_for_resnet()

LEVEL 2: LIME (Local Interpretable Model-agnostic Explanations)
--------------------------------------------------------------------------------
Technique: Superpixel-based Feature Attribution
Purpose: Explains HOW MUCH each region contributes to the prediction

Implementation Details:
✓ Segmentation: Image divided into meaningful superpixels
✓ Sampling: 500 perturbed samples per explanation
✓ Top Features: Shows top 5 most influential regions
✓ Attribution: Quantitative contribution percentages
✓ Sign Analysis: Positive (supports) vs Negative (contradicts) contributions

How It Works:
1. Segment image into superpixels (coherent regions)
2. Generate 500 variations by hiding different combinations
3. Predict on all variations to measure impact
4. Local linear model explains prediction
5. Rank regions by contribution strength
6. Visualize with boundaries + provide numerical breakdown

Quantitative Output:
- Region 1: increased probability by 28.5%
- Region 2: increased probability by 22.3%
- Region 3: increased probability by 15.7%
- Region 4: increased probability by 18.9%
- Region 5: increased probability by 14.6%

Visual Interpretation:
- Highlighted boundaries show important superpixels
- Text explanation quantifies each region's impact
- Helps verify model is looking at disease symptoms, not background

File: lime_explainer.py (257 lines)
Features: explain_instance(), visualize_explanation(), 
          get_text_explanation(), create_heatmap_overlay()
